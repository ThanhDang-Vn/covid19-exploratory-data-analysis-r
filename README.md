# COVID-19 Data Analysis with R

## Description

Exploratory data analysis of COVID-19 data using **R**, focusing on **testing numbers**, **positive case ratios**, and **data visualization** at the country level.

This project demonstrates how to clean, aggregate, analyze, and visualize real-world data using R and popular data science libraries.

---

## Objectives

* Read and explore COVID-19 datasets
* Aggregate data by country
* Identify top countries by:

  * Total testing numbers
  * Positive case ratios
* Store key findings using vectors and matrices
* Visualize insights using charts

---

## Tools & Libraries

* **R**
* **dplyr** – data manipulation
* **ggplot2** – data visualization
